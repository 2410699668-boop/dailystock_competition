# 恢复功能说明

## 选手总结

选手总结页新增：

1. 自动生成的选手风格画像；
2. 风格标签和画像可信度；
3. 个人累计收益折线图；
4. 最大回撤、收益波动、主要板块等指标；
5. 板块偏好和近期比赛记录。

只有正式结算的比赛日会进入累计收益曲线。临时收益不会进入正式曲线。

## 分析全部

每日选股右上角新增“分析全部”按钮。按钮会分析当前选择比赛日的所有选股：

- 同时最多分析 2 只，降低行情接口压力；
- 显示完成数量、成功数量和失败数量；
- 全部结束后一次性保存到 Supabase；
- 某只股票失败不会中断其他股票；
- 失败原因通过“错误！”按钮查看。

## 重新部署

将本项目全部文件覆盖上传到 GitHub 后，在 Netlify 执行：

```text
Deploys → Trigger deploy → Clear cache and deploy site
```
